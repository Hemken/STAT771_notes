Exercise 91 Proof theorem 11. Why does \(D\) have to be closed? (Hint: need \(\lim_{k \to \infty} x_k \in D\))

Solution. 1. Assume \(x^*\) and \(x_0\) satisfy \(x^* = G(x^*)\) and \(G(x_0) = x_0\). Then there exists an \(\alpha \in (0,1)\) such that \({\left \vert \left \vert G(x^*) - G(x_0) \right \vert \right \vert} \le \alpha{\left \vert \left \vert x^* - x_0 \right \vert \right \vert}\). But \(\alpha {\left \vert \left \vert x^* - x_0 \right \vert \right \vert} = \alpha{\left \vert \left \vert G(x^*) - G(x_0) \right \vert \right \vert}\), so \({\left \vert \left \vert G(x^*) - G(x_0) \right \vert \right \vert} \le \alpha {\left \vert \left \vert G(x^*) - G(x_0) \right \vert \right \vert}\). Since \(\alpha \in (0,1)\), this can only hold if \({\left \vert \left \vert G(x^*) - G(x_0) \right \vert \right \vert} = 0\), so \(x^* = G(x^*) = G(x_0) = x_0\).

  1. For some \(\alpha \in (0,1)\):

\[\begin{aligned} {\left \vert \left \vert G(x_{k+1}) - x_{k+1} \right \vert \right \vert} &= {\left \vert \left \vert G(x_{k+1}) - G(x_k) \right \vert \right \vert} \\ &\le \alpha{\left \vert \left \vert x_{k+1}-x_k \right \vert \right \vert} \\ &= \alpha{\left \vert \left \vert G(x_k)-G(x_{k-1}) \right \vert \right \vert} \\ &= \alpha^2{\left \vert \left \vert x_k-x_{k-1} \right \vert \right \vert} \\ &\vdots \\ &= \alpha^k{\left \vert \left \vert x_1-x_0 \right \vert \right \vert}. \end{aligned}\]

Hence, \({\left \vert \left \vert G(x_k - x_k) \right \vert \right \vert} \to 0\) as \(k \to \infty\), so \(\lim x_k = \lim G(x_k)\), i.e. \(\lim x_k = x^*\).

Exercise 93 Let \(G(x) = 0.5(x+\frac{4}{x})\). Solve \(x = G(x)\) by hand. What is \(G(x)\) doing?

Solution. \[\begin{aligned} G(x) &= x \Rightarrow \\ 0.5(x + \frac{4}{x}) &= x \Rightarrow \\ -0.5x + \frac{2}{x}) &= 0 \Rightarrow \\ -0.5x^2 + 2 &= 0 \Rightarrow \\ x^2 &= 4 \Rightarrow \\ x = \pm 2 \end{aligned}\]

Exercise 95 What is the impact of the singular values of \(J(x^*)\) on the “localness” of the results in theorem 12?

Solution. As seen in theorem 12, \({\left \vert \left \vert x_c - x^* \right \vert \right \vert} \le \min\left(\rho^*, \frac{1}{2\gamma {\left \vert \left \vert J(x^*)^{-1} \right \vert \right \vert}}\right)\). Assuming we are using \({\left \vert \left \vert \cdot \right \vert \right \vert}_2\), \({\left \vert \left \vert J(x^*)^{-1} \right \vert \right \vert}_2 = \frac{1}{\sigma_n}\), where \(\sigma_n\) is the largest singular value. So, a larger maximum singular value means the result is “more local”.

Exercise 97 Proof and interpret the following:

Let \(A,B \in {\mathbb{R}}^{n \times m}\) and assume \(A\) is non-singular. If for some \(\epsilon \in (0,1)\), \({\left \vert \left \vert AB - I \right \vert \right \vert} < 1 - \epsilon\), then \(B\) is invertible and

  1. \({\left \vert \left \vert A - B^{-1} \right \vert \right \vert} < (1-\epsilon){\left \vert \left \vert B^{-1} \right \vert \right \vert}\),
  2. \({\left \vert \left \vert B^{-1} \right \vert \right \vert} \le \frac{1}{\epsilon} {\left \vert \left \vert A \right \vert \right \vert}\).

Solution. Note that \(\sum_{k=0}^N (AB-I)^k AB = (AB-I)^N AB + AB\). Since \({\left \vert \left \vert AB-I \right \vert \right \vert} < 1-\epsilon\), \(\sum_{k=0}^\infty (AB-I)^k AB = AB\), so \(\sum_{k=0}^\infty (AB-I)^k\) is the inverse of \(AB\). Let \(D\) denote that sum. Then \(AB = D^{-1}\), which implies \(B=(DA)^{-1}\). So \(B^{-1} = DA\). Hence, \(B\) is invertible.

Furthermore, \({\left \vert \left \vert A-B^{-1} \right \vert \right \vert} = {\left \vert \left \vert (AB-I)B^{-1} \right \vert \right \vert} \le {\left \vert \left \vert AB-I \right \vert \right \vert}{\left \vert \left \vert B^{-1} \right \vert \right \vert} < (1-\epsilon){\left \vert \left \vert B^{-1} \right \vert \right \vert}\), and

\[\begin{aligned} {\left \vert \left \vert B^{-1} \right \vert \right \vert} &\le {\left \vert \left \vert A \right \vert \right \vert} {\left \vert \left \vert D \right \vert \right \vert} \\ &\le {\left \vert \left \vert \sum_{k=0}^\infty {\left \vert \left \vert AB - I \right \vert \right \vert}^k \right \vert \right \vert} \\ &< {\left \vert \left \vert A \right \vert \right \vert} \sum_{k=0}^{\infty} (1- \epsilon)^k \\ &= {\left \vert \left \vert A \right \vert \right \vert}\frac{1}{1-(1-\epsilon)} = \frac{1}{\epsilon}{\left \vert \left \vert A \right \vert \right \vert}, \end{aligned}\]
