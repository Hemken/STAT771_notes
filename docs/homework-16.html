Exercise 109 Let \(F: {\mathbb{R}}^m \to {\mathbb{R}}^n\) be everywhere differentiable such that in any closed ball, the derivative is bounded.

Show that in any closed ball, \(F\) is Lipschitz continuous.

Solution. Use Taylors theorem and let \(L = \sup_{\omega \in B}||\nabla F(\omega)||\):

\[F(x) = F(y) + \nabla F(y + \gamma(y-x))^T(y-x),\,\, for\,\, some\,\, \gamma \in (0,1)\]

\[\begin{aligned} ||F(x)-F(y)|| &= ||\nabla F(y + \gamma(y-x))^T(y-x)|| \\ & \leq ||\nabla F(y + \gamma(y-x))||\,||(y-x)|| \\ & \leq \sup_{\omega \in B}||\nabla F(\omega)||\,||(y-x)|| \\ & \leq L||(y-x)|| \end{aligned}\]

Exercise 110 Is \(f(x) = |x|\) semi-smooth?

Solution. Yes.

\(f\) is locally Lipschitz for all \(x \in {\mathbb{R}}\). For all \(x \neq 0\), we can find a ball such that for all \(y \in B\), \(\text{sign}(y) = \text{sign}(x)\). So, if \(x > 0, y,z \in B\), \(|f(z) - f(y)| = |z - y|\). If \(x < 0, z,y \in B\), \(|f(z) - f(y)| = |-z + y| = |z-y|\).

If \(x = 0\), we can find \(z,y \in B\) such that \(\text{sign}(z) \neq \text{sign}(y)\). Assume, without loss of generality, that \(z < 0 < y\). Then, \(|f(z) - f(y)| = |-z - y| \le |z - y|\). So, in any case, \(f\) is Lipshitz.

To verify second condition, let \(\epsilon > 0\). If \(x \neq 0\), pick a ball \(B\) around \(x\) such that \(\text{sign}(y) = 1\) for all \(y \in B\). If \(x > 0\), \(\partial f(u) = 1\) for all \(u \in B\). \({\left \vert \left \vert f(u) - f(x) - v(u-x) \right \vert \right \vert} = {\left \vert \left \vert u-x-u+x \right \vert \right \vert} = 0\) (\(v = 1\)). If \(x < 0\), \(\partial f(u) = -1\) for all \(u \in B\). Then \({\left \vert \left \vert f(u) - f(x) - v(u-x) \right \vert \right \vert} = {\left \vert \left \vert -u+x+u-x \right \vert \right \vert} = 0\) (\(v = -1\)). If \(x=0\), any ball around \(x\) will contain both positive and negative values. In either case, \({\left \vert \left \vert f(u) - f(x) - v(u-x) \right \vert \right \vert} = 0\) (\(v = \text{sign}(u), x=0\)).
