Exercise 4.7 Suppose \(R \in {\mathbb{R}}^{m\times m}\) is an upper triangular matrix with \(R_{ii} \neq 0\) for all \(i = 1,\ldots, n\). Is \(R\) invertible?
Solution. Since \(R\) is an upper triangular matrix, \(\det(R) = \prod_{i=1}^{m} R_{ii} > 0\). Hence, \(R\) is invertible.
